## 📁 Projetos e Exercícios Catalogados
### 1. Verificador de Idade Inteligente 🧟
* **Arquivo:** `verificador_de_idade.py`
* **Conceitos aplicados:** Variáveis (`str`, `int`), Condicionais (`if/else`), Laços de repetição (`while True`), e Tratamento de erros/Validação de dados (`try/except`).
* **O que faz:** O programa pergunta o nome e a idade do usuário. Ele possui uma proteção divertida para não aceitar letras no campo de idade e barra números absurdos (como idades maiores que 120 anos).
